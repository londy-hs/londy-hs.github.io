    hexo博客快速搭建

    描述

    Github Page + Hexo，快速搭建静态网页博客。

    准备环境

    1. Node.js

    2. Git

    3. 安装Hexo脚手架
      npm install -g hexo-cli || yarn add hexo-cli -g

    4. 博客项目初始化

      新建完成后，在路径下，会产生这些文件和文件夹：

    5. 启动服务器。输入以下命令，运行即可：

      hexo s

    6. 浏览器访问网址： http://localhost:4000/
      至此，您的Hexo博客已经搭建在本地。

    实施部署方案

    GithubPages

    1. 安装hexo-deployer-git插件 用于git仓库访问及上传

      yarn add hexo-deployer-git

    2. 添加SSH key

      ssh-keygen -t rsa -C "邮箱地址" // 创建ssh密钥

      生成的密钥地址，windows用户 是在 C:\Users\Administrator\.ssh\id_rsa.pub Mac用户是在 /Users/用户名/.ssh/id_rsa.pub

    3. 在gitHub-Settings-SSH and GPG keys 配置密钥地址

    4. GiHub-新建仓库

      Github 仅能使用一个同名仓库的代码托管一个静态站点

      另外，硬性要求：仓库名必须一定是 用户名.github.io,像我自己的就是 londy-hs.github.io

    5. 修改_config.yml，配置上自己git上的地址

      上面仓库地址写ssh地址，不写http地址。

    6. 推送到GithubPages
      hexo g -d
      浏览器输入: https://londy-hs.github.io

    GiteePages 同理

    主题选择

    Hexo默认的主题是landscape，主题选择： 官方主题

    每个主题可直接进入到相应的仓库中，按照主题配置说明和个性化需求，进行相应的定制化配置。

    插件

    评论 加入百度统计 网站错误分析等，插件系统

    1. 首先进项官网注册
    2. 创建应用
    3. 找到生成的key和id
    4. 在主题文件配置中进项配置
